# 毎日こんだてレビュー

人間中心設計の授業でユーザーレビューを行うための、スマホ向け料理献立アプリのプロトタイプです。

## 目的

忙しい大学生が、予算・調理時間・健康条件に合わせて献立を作り、買い物から調理、食後フィードバックまで進められるかを検証するための改善前プロトタイプです。

## 特徴

- 13画面のレビュー用フロー
- 外部API・ログイン・バックエンドなし
- 固定データによる献立、買い物リスト、代替食材、レシピ表示
- スマホ表示を想定したWebアプリ
